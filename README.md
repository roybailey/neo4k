# neo4k

Neo4j wrapper for Kotlin

## Installation

##### Maven 

```
    <dependency>
        <groupId>me.roybailey</groupId>
        <artifactId>neo4k-api</artifactId>
        <version>0.1.0-SNAPSHOT</version>
    </dependency>
```

##### Gradle 

`implementation("me.roybailey:neo4k-api")`
